Changing variable's values with out redeclare them

I currently find myself reviewing some basic JavaScript stuff following the schedule of a paid course. As the event started in the middle, I'll continue writing where I left off.

Despite being a subject that is believed to be little used, so far it has not been knew of the possibility:
In case it is necessary to exchange multiple variable values, the following method can be used.

Ex.: There are variables A, B and C; all with their respective values, now change the order of the variables without declaring them again. 

          
            let varA = 'A'; // Needs to be B 
            let varB = 'B'; // Needs to be C 
            let varC = 'C'; // Needs to be A

Therefore, it is possible to use square brackets ( [ ] ) to solve the problem:
          
            let varA = 'A'; 
            let varB = 'B'; 
            let varC = 'C';
          
            [varA, varB, varC] = [varB, varC, varA]


So now the variables have the values desired and did not need to be declared again.
OBS.: It is important point out that the values ​​in the second square bracket need to be in order with the values ​​to be changed.
