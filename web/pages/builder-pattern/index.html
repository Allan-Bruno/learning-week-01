Builder Pattern

The builder design pattern is a creational pattern that makes easy to construct complex objects, by implementing a step-by-step approach, in a way that each property of the object can be created separately, with the final object at the end.

In this example, we will build a car, step-by-step. Take a look at the table below. It displays the current info about the car, now try setting it's values, and click the "Build" button to see the result.
