Builder Pattern

O padrão de projeto builder é um pattern criacional que facilita a criação de objetos complexos, implementando uma abordagem passo-a-passo, de maneira que cada propriedade possa ser atribuída separadamente, tendo o objeto final no fim do processo.

Neste exemplo, criaremos um carro. Dê uma olhada na tabela abaixo. Ela guarda informações sobre o carro. Agora tente mudar os valores como desejar, e clique no botão "Construir" para ver o resultado.
